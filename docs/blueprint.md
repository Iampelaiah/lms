# **App Name**: LearnetIQ Frontend MVP

## Core Features:

- Role-Based Home Pages: Implement distinct landing pages and navigation for Students, Tutors, Parents, and School Admins, guiding users to relevant sections of the application.
- Student Dashboard: Create a personalized dashboard for students displaying their overall progress, quick access to subjects, and AI-powered learning recommendations from the AI Tutor Assistant using Genkit.
- AI Tutor Assistant: Integrate an AI tutor tool powered by Genkit on the Student Dashboard that can be prompted for learning resources based on student history. The tool intelligently decides what resources would be most helpful to the student.
- Course Browsing: Enable students to browse available subjects, view courses for each subject, and access course details including curriculum content.
- Community Forums: Provide a fully functional forum section where students can engage in discussions, create posts, view nested comments, and browse different communities.
- Resource Library: Implement a nested accordion-style interface for students to access and download learning resources categorized by subject and topic.
- Progress Tracking: Show student performance through visual charts and graphs
- Role-Based Dashboards: Implement distinct dashboards for Students, Tutors, Parents and School Admins

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent knowledge, trust, and modernity.
- Background color: A very light blue (#F0F8FF) to provide a clean and airy feel.
- Accent color: A saturated purple (#9C27B0) to highlight key interactive elements, adding a touch of sophistication and visual interest.
- Body font: 'Inter', a sans-serif font, to maintain a clean and modern look. Use for longer paragraphs.
- Headline font: 'Space Grotesk', a sans-serif font, giving a geometric and slightly futuristic look.
- Use modern and minimalist icons from a library like FontAwesome or Material Icons to ensure consistency and clarity across the interface.
- Employ a modular and responsive layout using Tailwind CSS to adapt seamlessly to different screen sizes, maintaining a consistent look and feel.
